* strict mode 사용
#### String.trim()
* String 공백 제거
#### Array.isArray()
* Object가 Array인지 확인
#### Array.forEach()
```
var list = [1,2,3,4,5]
list.forEach(printList);
function printList(value){
  console.log(value);
}
```  
#### Array.map()
```
var list = [1,2,3,4,5]
var list2 = list.map(plusOne);
function plusOne(value){
  return value+1;
}
```
